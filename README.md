# 🔷 VertexHub

Auto-type script for Roblox with draggable UI, console listener, and smart word suggestions.

## ✨ Features
- Draggable floating toggle button (logo V)
- Hitam-putih theme
- Auto-typing words from dictionary
- Console letter detection (supports game's "Word: XYZ" output)
- Smart word prioritization based on ideal length
- Pagination and word selection
- Auto-reset after 5 minutes inactivity

## 🚀 How to Use
1. Run the script in any Roblox executor that supports `VirtualInputManager`
2. Click the **V** button to show/hide the main UI
3. Type letters in the search box to get word suggestions
4. Click any word to auto-type it
5. Use side buttons: **Search, Listener, Longest, Reset, Settings**
